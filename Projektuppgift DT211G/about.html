Bakom webbplatsen

APIer

För webbplatsen används två APIer som hämtar information från olika platser, och sedan kombineras för att visa data på kartan.

WhereTheISSAt används för att hämta den Internationella Rymdstationens nuvarande plats som latitud och longitud.

OpenBreweryDB hämtar information om bryggerier så som namn, stad, latitud och longitud mm. Vanligtvis hade en formel, haversineformeln, krävts för att räkna ut avståndet men tack vare parametern "by_dist" räknar APIn själv ut avståndet mellan koordinater.

Karta

Som karta används Leaflet. Det är en open-source javascript kartfunktion som kan användas i samband med andra Geokod tjänster för att mäta avstånd och söka platser.

Grafiken för kartan hämtas från OpenStreetMap och implementeras via JavaScript. Det är en av många grafiker som man kan använda med Leaflet. Den är också gratis att använda, så länge man ger erkännande via deras länk.
