# DT211G - Projektuppgift

## [URL](https://projektuppgiftdt211g.netlify.app/)

Projekt skapat för DT211G - Frontend-baserad webbutveckling.<br>
Webbplatsen är till för törstiga astronauter som längtar efter en färsk, sval bira från fatet. Den hämtar information från [Open Notify](http://open-notify.org/Open-Notify-API/ISS-Location-Now/) som tillhandahåller den Internationella Rymdstationens position i latitud och longtid. <br>I samband med med [OpenBreweryDB](https://www.openbrewerydb.org/documentation#list-breweries) hittar den närmsta väg till ölen. 
<br>För kartan används [Leaflet](https://leafletjs.com/) och grafiken av [OpenStreetMap](https://www.openstreetmap.org/), med markörer och linjer som visuellt ger information om rymdstationen, avståndet och närmsta bryggeri.
