# DT211G - Projektuppgift

## [Apöllo 11](https://projektuppgiftdt211g.netlify.app/)
## [JSDoc](https://github.com/isasod2500/Projekt/blob/main/Projektuppgift%20DT211G/out/main.js.html)
Projekt skapat för DT211G - Frontend-baserad webbutveckling.<br>
Webbplatsen är till för törstiga astronauter som längtar efter en färsk, sval bira från fatet. Den hämtar information från [Where_the_ISS_at](https://wheretheiss.at/w/developer) som tillhandahåller den Internationella Rymdstationens position i latitud och longitud. <br>I samband med med [OpenBreweryDB](https://www.openbrewerydb.org/documentation#list-breweries) hittar den närmsta väg till ölen. 
<br>För kartan används [Leaflet](https://leafletjs.com/) och grafiken av [OpenStreetMap](https://www.openstreetmap.org/), med markörer och linjer som visuellt ger information om rymdstationen, avståndet och närmsta bryggeri.
